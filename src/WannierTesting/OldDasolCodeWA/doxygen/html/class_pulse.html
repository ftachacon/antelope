Propagation direction is dentermined by two paramter thetaz and phiz. They are just polar and azimuthal angle in spherical coordinatte, so propagation direction is . For the definition of LCP, and RCP I choose view from source. Therefore, for left or right hand rule, direction of thumb is same as propagtion direction (out of source). To doing standard 2d calculation (x-y plane), set thetaz = 0 and phiz = 0, or just omit them from the constructor.

generate envelope of pulse

A(t) = A0 f(t) cos(w0t - cep) = E0/w0 f(t) cos(w0t - cep), E(t) = -dA(t)/dt = - E0/w0 f'(t) cos(w0t - cep) - E0 f(t) sin(w0t - cep).

Parameters
_ttime. envelope has maximum value when _t=0. Therefore, t0 has to be subtracted before the function.
_isDtif true return time derivative of envelope. if false return envelope
